Rint

Description

Runtime integers retain much of their functionality while being reduced to the 32-bit, [-2147483648,2147483647] bounds.

Declaration

Rint var = 0;

Bounds

[-2147483648,2147483647]

Operators

//increment
var++
++var

//decrement
var--
--var

//assignment operators
var = n //set variable equal to n
var += n //add n to var
var -= n //subtract n from var
var *= n //multiply var by n
var /= n //divide var by n
var %= n //modulo var by n
var ^= n //raise to the power of an integer n
var ?= n //round to the nearest n

//expression operators
var + n //increased by n
var - n //subtracted by n
var * n //multiplied by n
var / n //divided by n
var % n //mod n
var ^ n //to the power of n
var !^ n //nth root
!^ var //sqrt
~ var //sin
!~ var //cos
var ? n //round to the nearest n (123 ? 10 is equal to 120)
