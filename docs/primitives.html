Primitives

Ugh ugh

Every primitive has both a Runtime and Compiletime variant. Compiletime primitives can be cast as Runtime primitives, but not the other way around.
As you might expect, runtime variables and compiletime variables do not mix.
Rint runtimeVariable = 1;
Cint compiletimeVariable = 2;

print(runtimeVariable + compiletimeVariable);
//this isn't allowed

print(runtimeVariable + Rint(compiletimeVariable));
//prints 3
//It takes a snapshot of the value of 'compiletimeVariable' and
//the compiler adds it to a list of constants that are declared when the datapack is loaded.
//compiletimeVariable remains a compiletime variable

compiletimeVariable ++;
print(compiletimeVariable = Rint(compiletimeVariable));
//compiletimeVariable is now of type Rint, and all operations done to it will happen at runtime.
//this is allowed because cint and Rint are both numbers and thus can be cast implicitly
