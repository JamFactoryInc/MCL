Cint

Description

Compiletime integers are based on python's integer, and thus are essentially unbounded.
They are also iterators, so you can iterate over a cint via a foreach loop.

Declaration

Cint var = 0;

Bounds

Realistically unbounded

Operators

//increment
var++
++var

//decrement
var--
--var

//assignment operators
var = n //set variable equal to n
var += n //add n to var
var -= n //subtract n from var
var *= n //multiply var by n
var /= n //divide var by n
var %= n //modulo var by n
var ^= n //raise to the power of an integer n
var ?= n //round to the nearest n

//expression operators
var + n //increased by n
var - n //subtracted by n
var * n //multiplied by n
var / n //divided by n
var % n //mod n
var ^ n //to the power of n
var !^ n //nth root
!^ var //sqrt
~ var //sin
!~ var //cos
var ? n //round to the nearest n (123 ? 10 is equal to 120)
